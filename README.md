# PCCT-Diffusion
Official repository for our upcoming paper on Score-based generative model framework with physical prior constraints for propagation-based X-ray phase-contrast CT reconstruction from incomplete projections.

### 📢 Status Update
The manuscript is currently under peer review. To protect the unpublished research while demonstrating our commitment to open science, we have established this repository. 

The complete source code, pre-trained models, and detailed dataset setup instructions will be fully released here immediately upon the paper's acceptance and official publication.
